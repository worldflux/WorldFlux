# Roadmap

This roadmap tracks the current technical program for WorldFlux.

## Current Priorities

1. Reference-fidelity DreamerV3 and TD-MPC2 surfaces
2. Public API stability and contract hardening
3. Parity evidence publication
4. Training lifecycle artifacts and verification tiers
5. Distributed training and replay scaling

## Maintainer Use

- Contributors should check this file before proposing broad new work.
- Maintainers should use this roadmap when prioritizing issues and release scope.
