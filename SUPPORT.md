# Support

Use the following support channels based on request type.

## Questions and Discussions

- Use GitHub Discussions: <https://github.com/worldflux/WorldFlux/discussions>
- Use Discord for community chat: <https://discord.gg/ZUBn9UEp2z>

## Bugs and Regressions

- Open an issue: <https://github.com/worldflux/WorldFlux/issues/new/choose>
- Include environment details and a minimal reproduction snippet.

## Security Reports

- Follow `SECURITY.md`.
- Report vulnerabilities privately to `yhyoda@worldflux.ai`.

## Response Expectations

- Security reports: initial response within 48 hours.
- Maintainer-managed issues and discussions: best-effort review in 3 business days.
